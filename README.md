# Mama Cherry's Pet Pawlor Website

This is a static website package ready for GitHub Pages.

## Files

- `index.html` - website content
- `styles.css` - colors, layout, and mobile styling
- `script.js` - mobile menu and email appointment form
- `assets/` - logo and placeholder images

## Before Publishing

Update these placeholders:

1. Phone number in `index.html`
2. Email address in both `index.html` and `script.js`
3. Business address in `index.html`
4. Facebook and Instagram links in `index.html`
5. Google Maps embed location in `index.html`
6. Service pricing if needed
7. Replace placeholder images in `/assets`

## GitHub Pages Deployment

1. Create a new public GitHub repository.
2. Upload all files from this folder into the repository root.
3. Go to Repository Settings > Pages.
4. Under Source, select `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.
7. After GitHub publishes the site, add your custom domain in Pages settings.
8. Create the DNS records in Squarespace Domains as GitHub instructs.
